This is the API

order

By passing the details of an order, the social proof for those products are updated, a product visit is counted once per session. The social proof for that product is returned

URL

https://virtserver.swaggerhub.com/Flockr_AI/Flockr/1.0.0/{apiKey}/conversion/order

HTTP Method

POST

Response Type

flockr

Parameters

  • Body: body Order(Order)

    Details of the product visit

  • Path: apiKey String

    your api key

  • Query: flockrDemo String

    append this parameter with the value \"true\" to return demo data for testing

  • Query: flockrFakeData String

    append this value to return demo data of a specific proof type
