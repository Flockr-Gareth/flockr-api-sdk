This is the API

categoryVisit

Returns a list of Social Proof values for the products provided

URL

https://virtserver.swaggerhub.com/Flockr_AI/Flockr/1.0.0/{apiKey}/category/visit

HTTP Method

POST

Response Type

flockr

Parameters

  • Body: body List(List)

    An array of product IDs or altIds

  • Path: apiKey String

    your api key

  • Query: flockrDemo String

    append this parameter with the value \"true\" to return demo data for testing

  • Query: flockrFakeData String

    append this value to return demo data of a specific proof type
