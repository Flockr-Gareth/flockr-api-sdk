This is the API

productVisit

By passing the details of a product visit, the social proof for that product is updagted, a product visit is counted once per session. The social proof for that product is returned

URL

https://virtserver.swaggerhub.com/Flockr_AI/Flockr/1.0.0/{apiKey}/product/visit

HTTP Method

POST

Response Type

flockr

Parameters

  • Body: body Product(Product)

    Details of the product visit

  • Path: apiKey String

    your api key

  • Query: flockrDemo String

    append this parameter with the value \"true\" to return demo data for testing

  • Query: flockrFakeData String

    append this value to return demo data of a specific proof type
